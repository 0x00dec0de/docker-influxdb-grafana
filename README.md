# Docker-compose files for a simple InfluxDB + Grafana stack

Run your stack:

```
git clone https://github.com/nicolargo/docker-influxdb-grafana.git
cd docker-influxdb-grafana
docker-compose up -d
```

Show me the logs:

```
docker-compose logs
```


Stop it:

```
docker-compose stop
docker-compose rm
```
